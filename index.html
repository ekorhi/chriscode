Common CSS properties

Common CSS properties are the ones that are oftens used by inexperience developer

Name Description Expected Values
text-align The text-align property will align text to the element that holds it, otherwise known as its parent. text-align value
font-weight The font-weight property controls how bold or thin text appears. font-weight value
color and background-color Foreground color is the color that an element appears in. For example, when a heading is styled to appear green, the foreground color of the heading has been styled. Conversely, when a heading is styled so that its background appears yellow, the background color of the heading has been styled. color and background-color value

Text-Align Property Value

The text-align property can be set to one of the following commonly used values:

Value Description
Left Aligns text to the left side of its parent element, which in this case is the browser.
Center Centers text inside of its parent element.
Right Aligns text to the right side of its parent element.
Justify Spaces out text in order to align with the right and left side of the parent element.

Font-Weight Property Value

The font-weight property can be set to one of the following commonly used values:

Value Description
Bold This will make text to appear bold on web page.
Normal This will make text to appear thin on web page. It is the default value.

Color and Background-Color Value

The color and background-color value can be set in three different ways:

Colors Format Description Examples
Name Colors English words that describe colors, also called keyword colors red, blue, black, yellow, gold, pink, green
RGB Colors Numeric values that describe a mix of red, green, and blue. #ffffff, #000000, #cccccc, #346564
HSL Colors Numeric values that describe a mix of hue, saturation, and lightness. hsl(123, 40%, 30%), hsl(220, 70%, 50%), hsl(103, 19%, 22%), hsl(139, 10%, 90%)
